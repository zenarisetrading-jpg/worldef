# Typography System

## Heading Sizes

| Level | Size | Weight | Line Height | Usage |
|-------|------|--------|-------------|-------|
| **H1** | 42-48px | Bold (700) | 1.1 | Page hero titles |
| **H2** | 32-36px | Bold (700) | 1.2 | Section headings |
| **H3** | 20-24px | Semibold (600) | 1.3 | Card titles, subsections |
| **H4** | 18px | Semibold (600) | 1.4 | Feature titles |

## Body Text

| Type | Size | Weight | Line Height |
|------|------|--------|-------------|
| **Body Large** | 18px | Regular (400) | 1.6 |
| **Body Default** | 16px | Regular (400) | 1.6 |
| **Body Small** | 14px | Regular (400) | 1.5 |
| **Caption** | 12px | Regular (400) | 1.4 |

## Metric Numbers

| Type | Size | Weight |
|------|------|--------|
| **Hero Metric** | 36-48px | Bold (700) |
| **Card Metric** | 28-32px | Bold (700) |
| **Inline Metric** | 20-24px | Semibold (600) |

## Font Family

```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
```

## Guardrails

- ❌ **MAX SIZE: 48px** - Nothing larger, ever
- ❌ NO ultra-thin weights (< 400)
- ❌ NO all-caps for body text
- ✅ Keep sizes modest and readable
- ✅ Consistent spacing between elements
