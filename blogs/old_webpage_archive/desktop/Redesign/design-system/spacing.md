# Spacing System

## Section Spacing

| Element | Value | Usage |
|---------|-------|-------|
| **Section Padding** | 80px vertical | Top/bottom of page sections |
| **Section Gap** | 0px | Sections stack directly |
| **Container Max Width** | 1200px | Content container |
| **Container Padding** | 24-32px | Horizontal padding |

## Card Spacing

| Element | Value |
|---------|-------|
| **Card Padding** | 40-60px |
| **Card Gap** | 24-32px |
| **Card Border Radius** | 12-16px |

## Element Spacing

| Element | Value |
|---------|-------|
| **Heading to Content** | 16-24px |
| **Paragraph Gap** | 16px |
| **Button Spacing** | 12-16px |
| **Icon to Text** | 12px |
| **List Item Gap** | 8-12px |

## Grid Gutters

| Breakpoint | Gutter |
|------------|--------|
| Desktop (>1024px) | 32px |
| Tablet (768-1024px) | 24px |
| Mobile (<768px) | 16px |

## Guardrails

- ❌ NO extreme spacing (>120px)
- ❌ NO cramped layouts (<16px gaps)
- ✅ Generous but reasonable white space
- ✅ Consistent 8px baseline grid
